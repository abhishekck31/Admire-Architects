# Admire Architects

A world-class, enterprise-grade architecture firm website built with:
- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP
- Lenis Smooth Scrolling

## Features
- Cinematic Parallax Hero
- Horizontal Execution Timelines
- Infinite Client Marquees
- Dynamic Project Masonry Grids
- GSAP Image Masking
- Custom Interactive Cursor
